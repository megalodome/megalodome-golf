## Why Vista here

## Account setup (30–45 min)
1. Log into Vista Social.
2. Connect brand-owned profiles (minimum):
   - Instagram
   - Facebook Page
   - LinkedIn Company
   - TikTok (optional)
   - YouTube (optional)
3. Upload brand kit to media library:
   - Logo, dome/hero stills, gold/black palette notes
   - `#EEDCA7` / `#0A0A0A`
4. Create labels: `raise-safe`, `venue`, `partners`, `community`, `press`.

## MCP setup (Hermes / Claude / Cursor)
From Vista: **Integrations → MCP**
1. Generate MCP connection for your AI client.
2. Connect Claude Desktop and/or Cursor.
3. Test prompts:
   - “Draft 5 Instagram captions for MEGALODOME indoor golf Oswego, non-solicitation tone.”
   - “Schedule a LinkedIn post tomorrow 9am CT linking to https://megalodomegolf.com/invest with UTM utm_source=linkedin&utm_campaign=awareness.”
   - “Summarize last 7 days engagement on Instagram.”

Do **not** ask MCP to publish PPM terms, share amounts, or “invest now / guaranteed returns” language.
